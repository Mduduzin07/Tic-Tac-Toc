üéØ Medium AI

Strategy: Mixed intelligence

Implementation: 50% smart moves, 50% random

üéØ Hard AI

Strategy: Perfect play

Implementation: Minimax algorithm

Use Case: Advanced challenge

Minimax Algorithm Key Points

  • Recursive depth-first search through all possible game states
  • Score assignment: +10 (AI wins), -10 (player wins), 0 (draw)
  • Backtracking: Propagates scores up the decision tree
  • Optimal move: Selects move with highest minimum guaranteed score
‚ö†Ô∏è Performance Note: Minimax can be computationally expensive for larger games, but for 3x3 Tic Tac Toe, it's perfectly efficient.

‚ö° Game Logic & Flow

Win Condition Detection

Approach: Pre-defined winning combinations array

const lines = [ [0,1,2], [3,4,5], [6,7,8], // Rows [0,3,6], [1,4,7], [2,5,8], // Columns [0,4,8], [2,4,6] // Diagonals ];

Game Flow Sequence

  1. Player clicks cell ‚Üí handleClick()
  2. Update board state and check for winner
  3. If game continues, set isPlayerTurn = false
  4. useEffect triggers AI move after 500ms delay
  5. AI calculates move based on selected difficulty
  6. Update board and check for winner again
  7. Repeat until game over condition met

üöÄ Performance & Optimization Notes

  • Move delay: 500ms delay makes AI feel more natural
  • Early termination: Win/draw detection stops unnecessary calculations
